# Sistema de Gestion para Consultorios Medicos Colombianos Epione

## Que Soluciona Epione?
Epione permite:

- Generar RIPS
- Agendar Citas
- Realizar Consultas Medicas
- Organiza las Finanzas del Consultorio

## Como lo Soluciona?

Epione esta organizado en dos modulos los cuales son modulo Medico y Administrador.

En el modulo medico estan las funciones de:
- Realizar Consulta medica

En el modulo Administrador estan las funciones de:
- Agendar Cita a Medico
- Generar RIPS de cada Medico o del Consultorio
- Registrar y Gestionar Finanzas
- Consultar Estadisticas de productividad del Medico y Consultorio
